My old snippets of code. Beware! It might make your eyes bleed. They are here for personal reference mainly.

You will need the Geant4 simulation libraries and the ROOT command interpreter installed in order for them to work.

Folders in this repo:

  • sim1: A simulation of a simple gamma detector with the Geant4 libraries.
    • Check Set up: Installing Geant4 and setting environment variables.
    • Check Simulation code: Description of the simulation code and results.
  • sim2: A simulation of an active target with the Geant4 libraries. Includes three types of calculations:
    • Bragg Curves: Calculate the range of a particle in a medium using the Bragg Curve inflection point.
    • Kinematics fortran: A FORTRAN routine to calculate the angles and energies of a particle dispersed in a medium.
    • Kinematics and simulation: Run a kinematics batch inside the simulation.
  • sim3: A stabilization function for photo-multipliers coupled to NaI scintillators.
